speckit 関連ドキュメントはすべて日本語で記述すること。

**Why:** ユーザーが明示的に指示した。プロジェクトの作業言語が日本語のため。

**How to apply:** `/speckit-*` コマンド実行時に生成・更新する `.specify/memory/` および `specs/` 配下のすべての Markdown ファイル（constitution.md, spec.md, plan.md, tasks.md 等）を日本語で記述する。テンプレートファイル（`.specify/templates/`）も日本語で更新する。
