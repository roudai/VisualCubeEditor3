## メモ

- 本仕様は spec 001（ロジックレイヤー）と spec 002（描画レイヤー）に依存する。
  両方完成済みであることを前提とする。
- Undo/Redo・スクランブル生成・キーボードショートカット・クラウド同期はスコープ外。
  Constitution v1.2.1 の優先機能リストに含まれない。
- LocalStorage の永続化は Constitution 原則VI の「作成状態の永続化」要件を満たす最小実装。
- svgdom / happy-dom の競合は実装上の技術制約（plan.md に記録）。spec には含めない。
